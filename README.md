# go-pedersen-hash

This library implements pedersen hashes in Go. It is based on this [reference doc](https://iden3-docs.readthedocs.io/en/latest/_downloads/4b929e0f96aef77b75bb5cfc0f832151/Pedersen-Hash.pdf) and fully compatible with [circomlib](https://github.com/iden3/circomlib/blob/e387ad83adef12ddd9b8bd91e225c5442891da21/src/pedersenHash.js).
